Reconhecimento Passivo

É a reunião de informações úteis extraídas à base da não agressão.

Nesta fase, temos como objetivo coletar dados sobre o alvo sem alertá-lo sobre nossa intenção, isso é, sem deixar com que ele saiba da nossa presença.

Tendo isso em mente, podemos usar de técnicas de reconhecimento que façam pouca, ou nenhuma, requisição ao alvo. Algumas delas são:

OSINT

Open Source Intelligence é a prática de coletar e analisar informações de fontes públicas, disponíveis na surface web. Útil para obter dados sem precisar acessar algo confidencial ou privado.

Google Hacking:

Syntax Exemplo Função
site site:scanme.nmap.org Pesquisar um site específico.
related related:scanme.nmap.org Pesquisar sites relacionados.
"" "phrase or word" Encontre resultados com a frase exata da pesquisa.
filetype filetype:pdf Encontre arquivos PDF relacionados a um determinado termo.
inurl inurl:tango.com Pesquisar URLs com palavras específicas.
intext intext:gmail.com Pesquisar palavras específicas no código-fonte.

Extração de domínio:
site:*.scanme.nmap.org

Operadores juntos:
site:*.scanme.nmap.org inurl:login filetype:txt

Retirada de email de vazamento:

É possível utilizar ferramentas como "have i been pwned" para ver se houve vazamento de uma determinada conta. A partir disso, baixando bancos de dados disponíveis na web, há ferramentas que extraem informações do BD.

Ex: grep -H/-a <arquivo_do_BD>

  • -H: Mostra qual arquivo é
  • -a: Muda o arquivo caso dê erro com binário

Shodan:

O Shodan é um site que recolhe informações de dispositivos conectados à internet. Usando o IP do server ou o DNS com a ferramenta DNSrecon, podemos usar o Shodan para usufruir disso.

Ferramentas

DNSrecon:

Serve para buscar subdomínios concatenados a um site ou até mesmo fazer um brute force.

Wappalyzer:

É uma extensão do navegador onde é possível descobrir frameworks front/back-end que o alvo utiliza no site, além de SO, versão de sistema. Podendo explorar vulnerabilidades já conhecidas.
